## 概要
- Git Large File Storage (LFS)は、オーディオサンプル、ビデオ、データセット、グラフィックなどの大きなファイルをGit内部でテキストポインタに置き換え、GitHub.comやGitHub Enterpriseのようなリモートサーバーにファイルコンテンツを保存します。
